## Übung(Directories und Files)
Aufgabe 2:
```
Lege mit mkdir und touch folgende Verzeichnisstruktur an:

./
└── abcd/
    ├── first_dir/
    │   ├── abcd.01.1.txt
    │   ├── abcd.01.2.txt
    │   └── abcd.01.3.txt
    └── second_dir/
        ├── xyz.02.1.txt
        ├── xyz.02.2.txt
        └── xyz.02.3.txt


```
 Lösung: 
```sh
- mkdir abcd/
- cd abcd/
- mkdir first_dir/
- touch abcd.01.1.txt
- touch abcd.01.2.txt
- touch abcd.01.3.txt
- cd ..
- cd abcd/
- mkdir second_dir/
- cd second_dir/
- touch xyz.02.1.txt
- touch xyz.02.2.txt
- touch xyz.02.3.txt
```
```
Weiters:

Stelle das Arbeitsverzeichnis auf abcd/second_dir und erstelle (ohne cd ein weiteres Mal zu verwenden):
ein Unterverzeichnis in_first_dir in first_dir, und
darin wiederum eine Datei neu.txt
Endgültig sieht es dann so aus:
./
└── abcd/
    ├── first_dir/
    │   ├── in_first_dir/
    │   │   └── neu.txt
    │   ├── abcd.01.1.txt
    │   ├── abcd.01.2.txt
    │   └── abcd.01.3.txt
    └── second_dir/
        ├── xyz.02.1.txt
        ├── xyz.02.2.txt
        └── xyz.02.3.txt
Verwende den Befehl tree um (alternativ zu ls) die Ausgabe in obiger Form zu erhalten. Achtung: tree muss von REPL nachinstalliert werden, das geschieht automatisch nach der Eingabe von tree – dabei die erste dargestellte Möglichkeit auswählen.
```
Lösung: 
```sh
- mkdir ../first_dir/in_first_dir/
- touch ../first_dir/in_first_dir/neu.txt

├── first_dir
│   ├── abcd.01.1.txt
│   ├── abcd.01.2.txt
│   ├── abcd.01.3.txt
│   └── in_first_dir
│       └── neu.txt
└── second_dir
    ├── xyz.02.1.txt
    ├── xyz.02.2.txt
    └── xyz.02.3.txt
```

## Übung(Befehle - Bewegen und Kopieren)

Aufgabe 3: 
```
Informiere dich über die Befehle rm, rmdir, cp und mv. Probiere die Befehle aus und dokumentiere deine Erkenntnisse im Arbeitsbericht.
```
```sh
rm =Löschen einer Datei

Bsp: touch ü_datei.txt
     rm ü_datei.txt
```
```sh
rmdir = Löscht Ordner
Hinweis: Der Ordner darf keine Dateien oder Unterordner haben.

Bsp: mkdir ü_ordner
     rmdir ü_ordner
 ```
 ```sh
 cp = Erstellt eine Kopie der Originaldatei 

 Bsp : cp ü_datei berichte = Es wird eine Kopie der Datei "Ü_datei.txt" erstellt und die Kopie wird in den Ordner berichte kopiert.  
 ```
 ```sh
 mv = Bennent oder Verschiebt Ordner/Datein um 

 Bsp: mv ü_ordner ü2_ordner = Der Ordnername ändert sich jetzt von ü_ordner zu ü2_ordner

 Bsp: mv ü_ordner berichte = Der Ordner "ü_ordner" wird jetzt in den Ordner berichte kopiert
 ```
