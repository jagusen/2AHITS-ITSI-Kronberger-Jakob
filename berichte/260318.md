Im Anschluss wird überprüft ob der GNU Debugger da ist.

 ## 3.Schritt: 

 Im GNU Debugger wird das Programm jetzt analysiert um die Passworteingabe zu umgehen

 Mit dem Befehl "gdb ./pwcheck" wird der GNU Debugger gestartet. Der Befehl "disas main"   zeigt uns den Opcode. 

 Im Code suchen wir nach den Befehlen cmp und jump. 

 ```sh
 cmp=compare
 ->Cmp bedeutet compare und vergleicht die Passworteingabe mit dem Grundlegenden Passwort
 ```

 ```sh
 Jump->Dieser Befehl springt zu der angegeben Adresse. 
 ```

 ## 4.Schritt:

 Suche im Opcode nach dem Jump Befehl. Sobald er gefunden ist müssen wir die 6 Bytes des Befehls mit einen NOP Befehl austauschen

 ```sh
 NOP=macht nichts (Keine Änderung)
 ```
 Das Austauschen sieht folgendermaßen aus: 

 ```sh
set {unsigned char}0x557a2e3301c8=0x90 
```
0x90 weil -> Das ist der Hexcode von NOP
