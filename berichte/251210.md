```sh
Was passiert wenn bei head eine negative Zahl für die Anzahl der Zeilen verwendet wird? Was steht dazu in der manpage von head?

Antwort: Wenn man eine negative Zahl angibt dann wird alles aus der Datei ausgegeben außer der letze anteil den an eingeben hat.

Was in der Manpage steht:

-n, --lines=[-]NUM
     print the first NUM lines instead of the first 10; with the
     leading '-', print all but the last NUM lines of each file




```



```sh

seq= seq gibt eine Liste von Zahlen aus – mit Startwert, Endwert und optional einer Schrittweite.
```
