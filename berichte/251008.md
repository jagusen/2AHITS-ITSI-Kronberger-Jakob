# Grundlagen

- Optionen,shorthand,longhand
- Argumente
- `echo`
- History
- Tastenkürzel
- Filesystem
    -`ls`, `cd`, `pwd`
    - Arbeitsverzeichnis, working directory
    - Unterschied zwischen relativer Pfad/absoluter Pfad
    - `mkdir`
    - `rm`
    - `rmdir`

Dieser Befehl löscht, vom Wurzelverzeichnis alles

```sh
rm -r /
```

Dieser Befehl zeigt den Inhalt des Wurzelverzeichnisses.

```sh
ls /
````

```
bin   etc     io     media  opt   repl  sbin   sys  var
boot  home    lib    mnt    pid2  root  srv    tmp
dev   inject  lib64  nix    proc  run   store  usr
~/workspace$ ^C
```

Dieser Befehl gibt folgendes Wort aus das du nach echo schreibst

```sh
 echo 2ahits
```

```
2ahits
```
Dieser Befehl(shorthand) gibt die Gesamtgröße der Datein und Verzeichnisse im aktuellen Ordner an

```sh
ls -l
```
```
total 8
drwxr-xr-x 1 runner runner 118 Oct  8 07:04 berichte
-rw-r--r-- 1 runner runner 102 Sep 24 07:04 README.md
```
Dieser Befehl(longhand) ist dann das gegenteil von shorthand 

```sh
ls -format==long
```

```
 .replit, README.md, .., ., berichte, .git
```
```
Der Befehl "History" zeigt dir die letzen Eingaben an. Du benutzt ihn mit den Pfeiltasten.
```

Du kannst mit den Tastenkürzeln an den Anfang des Wortes oder an das Ende des Wortes springen. Du kannst die Eingabe auch abbrechen.

```sh
strg+a oder strg+e oder strg+c
```

Dieser Befehl zeigt den Inhalt des Ordners in dem du dich gerade befindest
```sh
ls
```
Dieser Befehl zeigt dir an in welchen Ordner du dich gerade befindest
```sh
pwd
```


Mit diesem Befehl kannst du in einen anderen Ordner gehen bzw Ordner wechseln
```sh
cd
```
Mit diesem Befehl verlässt du den Ordner wieder
```sh
cd ..
```
Mit diesem Befehl kannst du einen neuen Ordner erstellen
```sh
mkdir test
```
Mit diesem Befehl kannst du den Ordner wieder löschen
```sh
rmdir test
```
Mit diesem Befehl kannst du eine neue Datei erstellen
```sh
touch datei.md
```
Mit diesem Befehl kannst du sie wieder löschen
```sh
rm datei.md 
```
```
Ein absoluter Pfad beschreibt den vollen Weg zu einer Datei oder einem Ordner
```
```
Ein relativer Pfad beschreibt den Weg ausgehend von deinem aktuellen Verzeichnis
```
