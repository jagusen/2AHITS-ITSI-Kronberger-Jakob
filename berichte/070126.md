## 1.Schritt

Öffne in Replit ein Fenster mit der Bezeichnung `Git`. Gehe dann auf das Zahnradsymbol und verknüpfe dich mit deinen Github Anmeldedaten.

## 2.Schritt 

Lege dir in Github ein neues Repository an nach folgender Reihenfolge: /Klasse/Fach/Nachname/Vorname.Kopiere nun den Link des Repositorys und füge den Link in den Replit-Einstellungen(Zahnradsymbol) unter Remote ein. 

## 3.Schritt

Verlasse die Einstellungen und drücke auf das Feld wo `Stage and commit all changes` steht. Danach auf den darüberliegenden Kopf drücken wo `Sync Changes` steht.

## 4.Schritt

Gehe jetzt unter den Einstellungen des Repositorys auf `Pages` und wähle dort bei `Branch ` main aus und drücke auf save. Drücke nun auf Workflow und öffne dort den Link unter deploy im Diagramm.

## 5.Schritt

Um auf die einzelnen Berichte zu kommen muss man in Git hub unter Code auf den Bericht drücken und oben den Link kopieren. Nun musst du die Kopie hinter die URL der Website hineinkopieren und noch das .md durch ein .html ersetzen.

## 6.Schritt

Du kannst auch im README Links zu den Berichten machen das geht folgendermaßen: 

[Angezeiger Link als Text den du selbst eingibst](Link zum Bericht)
