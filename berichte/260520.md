### 2. 



- In welchen Varianten können Felder mit der -f Option angegeben werden?
- Was unterscheidet die -c Option von -f?

- f → Felder (Spalten)
- c → Zeichenpositionen

- Mehrere einzelne Felder:

```sh
cut -d: -f1,3 datei.txt
```
-> Feld 1 und 3

- Bereich von Feldern:

```sh
cut -d: -f1-2 datei.txt
```
-> Feld 1 - 2

- Ab einem Feld bis zum Ende:
```sh
cut -d: -f2-
```
-> Ab Feld 2 bis zum Ende

- Vom Anfang bis zu einem Feld

```sh
cut -d: -f-2
```
-> Von Feld 1 bis 2


### Unterschied zwischen `-c` und `-f`

`-f` -> Arbeitet mit Feldern, braucht oft `-d` zur Angabe , 

`-c` -> Arbeitet mit Zeichenpositionen, braucht kein Trennzeichen
