##  Neue Befehle:

Cat: Mit dem Befehl "cat" kannst du Dateien anzeigen, zusammenfügen und erstellen.

```sh
Bsp: cat >test.txt  
Ausgabe --> Du kannst dann eingeben was du in die Datei schreiben möchtest: hallo

Bsp: cat <test.txt 
Áusgabe --> hallo  

```

Echo: Der Befehl gibt Wörter auf dem Bildschirm aus, die du davor eingegeben hast

```sh
Bsp: echo hallo 2ahits >hallo.txt 
--> Der Text "hallo" wurde jetzt in die Datei "hallo.txt" geschrieben.

Bsp: echo ich bin jakob >>hallo.txt 
--> Jetzt wird das davor eingegebene  nicht überschrieben. 

Ausgabe: hallo 2ahits
         ich bin jakob
```

Quotes: Damit wird die Variable nicht ersetzt, sie bleibt.

```sh
Bsp: '!llo' 
--> Ohne diese zwei Quotes würde sich das Wort nicht ausgeben lassen.
```

